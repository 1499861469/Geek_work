# learn_git
## git
分布式版本控制软件
git init 初始化git项目
git config --global user.name "" 初始化用户名配置
git config --global user.email  "" 初始化邮箱配置
git config --global --list 列出配置文件
git status 查看git状态
git add 跟踪新增文件
git add . 将所有已修改的文件提交到暂存区
git commit -m "新增变更"
git branch -M 分支 切换分支
git remote add origin git仓库地址  添加远程仓库地址 origin 远程仓库代号
